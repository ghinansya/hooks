#
- Setup dan Install
Requirements :
  
    Node.js (npm)


Clone Repository
```html
git clone https://github.com/ghinansya/hooks.git
```
```ruby
cd/hooks
```

Menginstal package dan expo dalam proyek
```ruby
npm install
```

Menjalankan proyek
```ruby
npm start
or
npx expo start
```
Proyek biasanya berjalan di local dengan port 8081 dan url:
```ruby
http://localhost:8081
```
